# CatMemo
 - 配信者向けメモ帳アプリ（配信者じゃなくても使えるかも！）
 ## 仕様
 1. 右クリックで終了
 2. 左ダブルクリックで常に前面表示 or 後面に隠れる切り替え
 3. Shift + → フォントサイズを大きく
 4. Shift + ← フォントサイズを小さく
 5. 画像に向けてドラッグアンドドロップ（画像ファイル）で画像切り替え
 6. 文字入力でふきだしが自動調整
 7. 画像をドラッグするとフォームの位置変更
 ## ダウンロード
 KihetTools.zipをダウンロードし、解凍
 CatMemoフォルダのSetup.exeを起動
 →インストーラーが開きます
